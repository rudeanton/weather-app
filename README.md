Weather app🌤️

Simple weather app that fetches current weather data from OpenWeather API and displays it dynamically.

Features:

- Search weather by city
- Display temperature, humidity and description
- Dynamic weather emoji
- Responsive design for mobile devices

Technologies:

- HTML
- CSS
- JavaScript (Fetch API)

How to use:

1. Open index.html in your browser
2. Enter a city name on the input
3. Click "Get weather" to see the weather info
